## What is this?

**jv-ytdlp** is a local desktop app that runs on your Windows PC and lets you download YouTube videos through a clean browser UI ‚ÄĒ no command line needed after setup.

Paste a URL ‚Üí pick quality ‚Üí download. That's it.

## Features

- Smart quality detection ‚ÄĒ only shows resolutions the video actually has
- Real-time progress bar with speed, ETA, and fragment counter
- Auto-merges video + audio using ffmpeg into a clean `.mp4`
- Audio-only MP3 mode
- Cancel download mid-way
- Live log console
- Neo-blue UI
- One-click installer ‚ÄĒ downloads yt-dlp and ffmpeg automatically on first launch

## How it works

The app runs a local HTTP server on port `57329`. The Electron window loads the UI from that server. yt-dlp and ffmpeg are downloaded once to `~/.jv-ytdlp/tools/` on first launch and reused on every subsequent run.

## Troubleshooting

**Setup screen appears on every launch**
‚Üí yt-dlp or ffmpeg may have failed to download. Click Install Tools again.

**No audio in downloaded video**
‚Üí ffmpeg may not have installed correctly. Re-run setup from the app menu.

**Quality shows only "Best Available"**
‚Üí Format scan failed ‚ÄĒ download still works using best available quality.

**Windows SmartScreen warning on installer**
‚Üí Expected for unsigned apps. Click "More info" ‚Üí "Run anyway". The app is open source ‚ÄĒ inspect the code in the source zip if you'd like to verify.
